![architecture](images/architecture.jpeg)
*This diagram shows how your Mood-Based Productivity App works step by step. First, the user opens the website and enters how they feel (for example, ‚ÄúI feel tired‚Äù). When the user clicks the ‚ÄúAnalyze Mood‚Äù button, the frontend (React) sends this text to the Flask backend using a POST request to the /analyze API. The backend then processes the text, detects the mood (such as stressed, low energy, or motivated), and selects appropriate responses like a music link, a productivity plan, and a motivational tip. It sends this data back to the frontend in JSON format. The frontend then displays the detected mood, plan, tip, and music option to the user. If the user clicks ‚ÄúStart Timer,‚Äù the Pomodoro timer begins (usually 25 minutes). After the timer finishes, the session is complete.*
